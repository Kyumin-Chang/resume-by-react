 ##  구현한 기능

1. 기존에 html과 css로 제작한 디자인을 따릅니다.
2. 다크모드와 라이트모드를 추가하고 이를 변경할 수 있는 버튼을 추가합니다.
3. 프로젝트를 클릭 시, 상세 정보를 볼 수 있는 모달 창이 띄워질 수 있도록 합니다.
4. Media Query를 이용하여 반응형 디자인을 구현합니다.
